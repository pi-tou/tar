🔧 GUIDE DÉMONTAGE/REMONTAGE DES MEUBLES

Temps total estimé : 15-20 heures | Personnes nécessaires : 2 minimum pour gros meubles

⚠️ PRÉPARATION AVANT DÉMONTAGE

Matériel à préparer

  • Appareil photo/smartphone chargé
  • Sachets zip (30 minimum)
  • Marqueur permanent
  • Scotch papier pour étiquetage
  • Couvertures protection sol
  • Boîtes pour petites pièces
  • Notices meubles (imprimer si besoin)

Organisation espace

  • Zone démontage dégagée
  • Zone stockage pièces démontées
  • Protection sol avec cartons
  • Éclairage suffisant
  • Accès outils facile

🛏️ MEZZANINE ADULTE 220×220 cm

DÉMONTAGE (2h - 2 personnes OBLIGATOIRE)

Outils : Clé Allen 6mm, Clé 13, Tournevis

Phase 1 - Préparation (20min)

  • Photos générales 4 faces
  • Retirer matelas
  • Retirer lattes sommier
  • Photos détails assemblages
  • Numéroter poutres au crayon (P1, P2…)

Phase 2 - Démontage structure haute (40min)

  • Personne 1 : Maintient structure
  • Personne 2 : Dévisse
  • Retirer barrières sécurité
    • Vis dans sachet “MZ-BARR”
  • Démonter cadre sommier
    • Vis dans sachet “MZ-SOMM”
  • Séparer traverses longues
    • Vis dans sachet “MZ-TRAV”

Phase 3 - Démontage pieds (40min)

  • Dévisser pieds un par un
  • ATTENTION : Soutenir structure !
  • Pieds dans sachet “MZ-PIEDS”
  • Démonter échelle/escalier
    • Vis dans sachet “MZ-ECH”

Phase 4 - Finalisation (20min)

  • Regrouper poutres par taille
  • Film plastique autour poutres
  • Scotcher sachets vis sur pièce principale
  • Compter toutes les pièces
  • Photo inventaire final

REMONTAGE (3h - 2 personnes)

  • Installer pieds niveau sol
  • Monter structure base
  • Vérifier niveau à chaque étape
  • Assembler cadre sommier
  • Fixer barrières sécurité
  • Installer échelle
  • Test solidité avant matelas
  • Point sécurité : Resserrer après 1 semaine

⚠️ SÉCURITÉ : Ne jamais monter seul - Risque de chute structure

🛏️ LIT KURA MEZZANINE IKEA

DÉMONTAGE (1h - 1 personne possible)

Outils : Clés Allen IKEA, Tournevis cruciforme

Étapes démontage (60min)

  • Photos position actuelle (haute/basse)
  • Retirer matelas + lattes
    • Compter lattes : /
  • Dévisser barrières latérales
    • Sachet “KURA-BARR”
  • Démonter structure lit
    • Sachet “KURA-LIT”
  • Séparer montants verticaux
    • Sachet “KURA-MONT”
  • Démonter panneaux déco (si présents)
    • Sachet “KURA-DECO”
  • Conserver notice IKEA

REMONTAGE (1h30)

  • Suivre notice IKEA page par page
  • Option position : ☐ Haute ☐ Basse
  • Montants verticaux d’abord
  • Cadre lit
  • Barrières sécurité
  • Vérifier tous points vissage
  • Installer lattes (ordre spécifique)
  • Test solidité avant enfant

💡 Astuce : Position basse pour enfant 3 ans, haute vers 6 ans

📚 KALLAX 16 CUBES (147×147×39 cm)

DÉMONTAGE (30min)

Outils : Tournevis cruciforme, Marteau caoutchouc

  • Retirer tous objets/paniers
  • Photos face avant organisation
  • Retirer panneau arrière (clous)
    • Attention : Fragile !
  • Dévisser étagères horizontales
    • Sachet “KAL16-H”
  • Dévisser étagères verticales
    • Sachet “KAL16-V”
  • Séparer cadre extérieur
    • Sachet “KAL16-CADRE”
  • Emballer panneau arrière (fragile)

REMONTAGE (45min)

  • Sol parfaitement plat requis
  • Assembler cadre extérieur
  • Insérer divisions verticales
  • Insérer divisions horizontales
  • Équerre pour vérifier angles droits
  • Clouer panneau arrière
  • Fixer au mur (anti-bascule)

⚠️ ATTENTION : Très lourd une fois monté - 2 personnes pour déplacer

📚 KALLAX 12 CUBES (147×112×39 cm)

DÉMONTAGE OPTIONNEL (25min)

Peut rester monté si place camionnette

Si démontage nécessaire :

  • Même procédure que 16 cubes
  • Sachet “KAL12-H” horizontales
  • Sachet “KAL12-V” verticales
  • Sachet “KAL12-CADRE”

Transport monté

  • Film plastique intégral
  • Couvertures protection angles
  • Position couchée dans camionnette
  • Sangles arrimage obligatoires

👕 ARMOIRE KLEPPSTAD (176×118×55 cm)

DÉMONTAGE (45min)

Outils : Tournevis, Clé Allen

  • Vider complètement
  • Retirer portes
    • Numéroter gauche/droite
    • Sachet “KLEP-PORTES”
  • Retirer tringles/étagères
    • Sachet “KLEP-INT”
  • Si miroir : PROTECTION MAXIMALE
    • Papier bulle + carton
    • Marquage “FRAGILE”
  • Démonter structure
    • Panneaux latéraux
    • Haut/bas
    • Fond
  • Regrouper panneaux similaires

REMONTAGE (1h)

  • Base niveau impératif
  • Assembler caisson principal
  • Installer fond (renfort structure)
  • Tringles et supports étagères
  • Ajuster portes (charnières)
  • Réglage fin portes
  • Fixer anti-bascule mur

🪑 TABLE MARBRE OVALE (72×120×60 cm)

DÉMONTAGE (10min)

⚠️ FRAGILE - 2 personnes

  • Photos système fixation pieds
  • Retourner table sur couvertures
  • Dévisser 4 pieds
    • Sachet “TABLE-PIEDS”
  • PROTECTION PLATEAU :
    • Papier bulle 3 couches
    • Film plastique
    • Carton dur dessus/dessous
    • Marquage “FRAGILE - MARBRE”
  • Transport vertical si possible

REMONTAGE (10min)

  • Vérifier état plateau
  • Table retournée sur protection
  • Visser pieds dans ordre (opposés)
  • Retourner avec précaution
  • Ajuster pieds si bancal

📋 ORGANISATION SACHETS VIS

Système codage

Format : [MEUBLE]-[PARTIE]-[NUMÉRO]

Exemple : MZ-BARR-01 = Mezzanine - Barrières - Sachet 1

Par meuble

  • MEZZANINE : MZ-___
  • KURA : KURA-___
  • KALLAX 16 : KAL16-___
  • KALLAX 12 : KAL12-___
  • KLEPPSTAD : KLEP-___
  • TABLE : TABLE-___
  • BUREAU : BUR-___
  • SMÅSTAD : SMA-___

Stockage sachets

  • 1 boîte par meuble
  • Étiquette extérieure visible
  • Scotcher sur pièce principale
  • Photo contenu avant fermeture

✅ CHECK-LIST SÉCURITÉ

Avant démontage

  • Dégager zone travail
  • Éclairage suffisant
  • Outils en état
  • Protection sol
  • Aide disponible pour gros meubles

Pendant démontage

  • Photos à chaque étape
  • Vis immédiatement en sachets
  • Numérotation continue
  • Pause toutes les heures
  • Hydratation régulière

Après remontage

  • Resserrage tous points
  • Test solidité
  • Fixations murales
  • Anti-bascule installés
  • Resserrage J+7 obligatoire

📝 NOTES TECHNIQUES

Couple serrage recommandé :

  • Bois/Bois : Serré mais pas forcé
  • Métal/Bois : Ferme + 1/4 tour
  • Métal/Métal : Ferme + 1/2 tour

Ordre remontage général :

  1. Base/pieds
  2. Structure verticale
  3. Structure horizontale
  4. Renforts/traverses
  5. Finitions/portes
  6. Sécurités/anti-bascule
