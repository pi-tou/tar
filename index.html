🏷️ SYSTÈME DE CODAGE DES CARTONS

Code couleur par pièce

  • 🔴 Rouge : Chambre enfant
  • 🔵 Bleu : Chambre adulte
  • 🟡 Jaune : Cuisine
  • 🟢 Vert : Séjour
  • ⚪ Blanc : Salle de bain
  • ⚫ Noir : Entrée/Divers

Numérotation

Format : [Couleur]-[Numéro]-[Priorité]

Exemple : R-01-A (Rouge, carton 1, priorité A)

  • A : Ouvrir immédiatement
  • B : Semaine 1
  • C : Peut attendre
💡 Astuce : Photographier le contenu de chaque carton avant fermeture et créer un fichier Excel avec numéro + contenu

💡 ASTUCES SPÉCIALES ENFANT 3 ANS

🎯 Stratégies pour faciliter la transition

Préparer un "sac magique" de déménagement
Nouveaux petits jouets, autocollants, livres surprises à découvrir pendant le trajet
Faire participer l'enfant
Décorer son carton spécial, choisir la couleur de sa chambre, "aider" à emballer ses jouets
Maintenir les routines
Heures de repas, sieste, rituel du coucher identiques même dans le chaos
Créer un coin familier immédiatement
Installer ses peluches, sa veilleuse, ses livres dès l'arrivée
Prévoir des repas simples qu'il aime
Pâtes au beurre, jambon-purée, ses plats préférés en version facile
Explorer le nouveau quartier comme une aventure
Trouver le parc, la boulangerie, faire une chasse au trésor dans le nouvel appartement
